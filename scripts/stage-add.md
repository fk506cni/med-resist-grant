ステージング拡張コマンドを実行してください。

## 処理手順

1. **既存message.mdのアーカイブ**
   - `scripts/archive_message.sh` を実行して、既存の `message.md` を `jank/` フォルダにタイムスタンプ付きで移動

2. **変更のステージング**
   - 直前の作業で変更されたファイルを `git add` でステージング
   - ステージング対象は会話の文脈から判断すること

3. **コミットメッセージの出力**
   - ステージングした内容に基づいてコミットメッセージを作成
   - プロジェクトルート直下の `message.md` に出力
   - フォーマット:
     ```
     変更の要約（1行）

     - 変更点1
     - 変更点2
     ```

## 注意事項

- コミット自体は実行しない（ユーザーが行う）
- `message.md` は `.gitignore` に含まれているため、コミット対象外
- ステージング後に `git status` で結果を表示すること
